# Lemberg Coach Bot — V1 (Quote + Action)

Це MVP Telegram-бота, який надсилає щоденну **цитату + завдання** та має кнопки:
- 🔥 Отримати мотивацію
- ✅ Завдання дня
- (опційно) 🧭 Відкрити Mini App, якщо задаш URL у `.env`

> План: локальний тест на Mac → далі хостинг для 24/7 роботи.

---

## 1) Встановлення на Mac

1. **Встанови Python 3.11+** (у більшості Mac вже є 3.11). Перевір:
   ```bash
   python3 --version
   ```

2. Відкрий **Terminal** і перейдіть у папку з проектом:
   ```bash
   cd ~/Downloads  # або інша папка, куди ти завантажиш zip
   ```

3. **Створи віртуальне середовище** та активуй:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```

4. **Встанови залежності**:
   ```bash
   pip install -r requirements.txt
   ```

5. **Скопіюй `.env.example` у `.env`** і встав свій токен бота:
   ```bash
   cp .env.example .env
   # відредагуй файл .env у будь-якому редакторі і встав BOT_TOKEN=...
   ```

6. (Опційно) Вкажи `MINI_APP_URL=` у `.env`, якщо вже маєш посилання на свій WebApp.

---

## 2) Запуск локально

```bash
python bot.py
```

- У Telegram відкрий `@LembergCoachBot` → `/start`
- Натисни кнопки, перевір як приходять **цитата + завдання**
- Команди:
  - `/subscribe` — вмикає щоденну розсилку о 09:00 (Europe/Berlin)
  - `/unsubscribe` — вимикає розсилку

> ⚠️ Щоб щоденна розсилка працювала, скрипт має **бути запущений**.

---

## 3) Контент

- Файл `quotes.json` — 7 записів (цитата + дія). Можеш доповнювати.
- Логіка обирає запис **за днем** (щоб у всіх користувачів був однаковий "День N").

> Хочеш Google Sheets? Можна підключити пізніше. Для швидкого старту вистачає JSON.

## 5) Типові питання

**Бот не відповідає?**
- Перевір токен у `.env`
- Перезапусти `python bot.py` (активувавши `.venv`)

**Чи можна додати WebApp / Premium?**
- Так. Додамо після успішного тесту V1.

**Розсилка не прийшла о 9:00?**
- Скрипт не був запущений, або ти не натиснув `/subscribe`.
- Час налаштовується через `DAILY_HOUR` у `.env`.
